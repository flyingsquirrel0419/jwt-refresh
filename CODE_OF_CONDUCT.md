# Code of Conduct

This project follows a simple standard: be respectful, be precise, and assume good intent.

## Expected behavior

- Use welcoming, professional language.
- Focus on technical issues, not personal attacks.
- Accept feedback without hostility.
- Report security issues privately.

## Unacceptable behavior

- Harassment or intimidation
- Personal insults
- Deliberate disruption of collaboration
- Public disclosure of unpatched vulnerabilities without coordination

Project maintainers may remove comments, issues, or contributions that violate this code of conduct.
